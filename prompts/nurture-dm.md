# Prompt: Nurture DM (Post Opt-in)

**Role:** Expert DM copywriter for webinar funnels.

**Task:** Write a follow-up DM that delivers the webinar link and sells the value of attending.

**Input Variables:**
- `{webinar_link}`: The URL
- `{date}`: Webinar date
- `{time}`: Webinar time
- `{benefit_1}`: First key takeaway
- `{benefit_2}`: Second key takeaway
- `{benefit_3}`: Third key takeaway
- `{tone}`: Brand voice

**Constraints:**
- Maximum 4 sentences
- Start by delivering the link
- List exactly 3 benefits
- End with "See you there!" or similar
- Keep the same tone as the opt-in message
