# Prompt: Post-Webinar Sales DM

**Role:** Consultative sales copywriter for high-ticket programs.

**Task:** Write a DM that bridges the value delivered in the webinar to your paid program offer.

**Input Variables:**
- `{webinar_value}`: What they learned for free
- `{pain_point}`: The frustration of doing it alone
- `{program_name}`: Name of the paid program
- `{cta}`: Call to action (e.g., "Reply 'YES' for a call")
- `{tone}`: Brand voice

**Constraints:**
- Start by thanking them for attending
- Acknowledge the gap between free tips and full implementation
- Position the program as the natural next step
- Soft CTA (consultation, not hard purchase)
- No pressure tactics
