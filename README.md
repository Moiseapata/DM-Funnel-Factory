# 🚀 DM Funnel Factory – Automated Copy System for High-Ticket Webinars

**By [Votre Nom] – For School of Bots Interview**

## 💡 The Vision
This project demonstrates a complete, AI-augmented system for creating, adapting, and automating high-converting DM funnel copy. It combines **strategic marketing frameworks** with **AI prompt engineering** and **automation workflows** to deliver scalable, on-brand copy at the speed of light.

## 🧠 The Core Framework
At the heart of this system is a proven **4-step DM funnel strategy** (see `/framework/master-guide.md`), designed for webinar promotion. The framework defines the exact KPIs for each step, from opt-in to sales conversion.

### The 4-Step Sequence:
| Step 1: Opt-in DM | Step 2: Nurture DM | Step 3: Reminder DM | Step 4: Sales DM |
|:---:|:---:|:---:|:---:|
| Problem hook + CTA | Link + 3 benefits | Urgency + re-hook | Value bridge + offer |

## 🎨 The Brand Voice Matrix
A unique feature is the **Tone Adaptation Matrix** (see `/framework/tone-matrix.md`), which allows the same webinar concept to be rewritten for three distinct brand voices:
- **Prince EA**: Inspirational & storytelling
- **Nike**: Direct & action-oriented
- **Mindvalley**: Mysterious & spiritual

## 🤖 AI vs. Human: The Optimization Layer
I've included a critical comparison showing how raw AI output is transformed into high-converting copy (see `/prompts/ai-vs-human-optimizer.md`). This proves my value isn't just *using* AI, but *refining* it with human insight, specificity, and emotional intelligence.

## ⚙️ Automation & Scalability
This isn't just theory. The system includes:
- **Python scripts** to generate full funnels from a client brief.
- **Prompt libraries** for each step, versioned and ready to use.
- **A Zapier/Make workflow blueprint** that automates the entire process from Google Sheets to final copy delivery.

## 📂 Project Structure
- `/framework`: The strategic master guides.
- `/prompts`: Engineered prompts for AI copy generation.
- `/scripts`: Python automation to generate and adapt copy.
- `/examples`: Sample JSON outputs for different brands.
- `/automation`: No-code automation workflow designs.
